Definitions

Domain (canonical count)
Registered + deployed publication domains in the network. Each apex domain and each functionally distinct subdomain counts separately. The canonical figure (180+) is maintained in counters.json.
Live (HTTPS 200)
The apex returns HTTP 200 over HTTPS within a 10-second timeout. A domain can be registered and deployed but temporarily off-line for DNS or cert reasons; those are noted with explicit error states.
Sitemap URLs
Count of <loc> entries in the domain's /sitemap.xml. Empty when the sitemap is missing or uncrawlable.
SSL state
Result of the TLS handshake on port 443. days left is the cert expiry remaining.
GA4
Whether the page contains a googletagmanager.com/gtag/js?id=G- reference (canonical OBP analytics).

Methodology

The audit script (scripts/btn_health_refresh.py) walks every statute-sites/<domain>/CNAME file plus a small set of core hosts (OBP, 1328f.com, 1328f.org, section1328.org). For each domain it: probes http:// and https:// apex, reads the TLS cert expiry, fetches sitemap.xml and counts <loc> entries, and records page size, title, and presence of GA4 + BTN engage scripts. The full JSON output is checked into scripts/btn_health_state.json in our public repo.
