Methodology

A catalog of named methodology contributions developed in the field by the Open Bankruptcy Project. Each entry has its own canonical page with full provenance, first-deployment artifact, related contributions, and CC-BY-4.0 license terms intended for citation.

Overview

The Open Bankruptcy Project Methodology section publishes canonical pages for novel methodology contributions developed through applied work in federal bankruptcy oversight, attorney accountability, and pro-se litigation. Each contribution is a named, versioned artifact with a stable canonical URL, a documented first deployment, and a citation profile suitable for use by practicing attorneys, empirical legal researchers, journalists, and legal-technology practitioners.

The catalog reflects techniques surfaced (and named) during real-world deployments rather than abstract proposals. Provenance is preserved: every contribution links back to a first-deployment artifact in the source repository, the related contributions it pairs with, and the underlying memory or working file from which it was extracted. The catalog is updated as new techniques mature.

This is not a peer-reviewed academic series. OBP publishes the contributions openly so that the broader field can evaluate, critique, replicate, or extend the work. We welcome correspondence (research@openbankruptcyproject.org) regarding adoption, adaptation, or substantive challenge to any entry.

Adopted Contributions

Two prior OBP methodology contributions have already been accepted by the federal judiciary through the Advisory Committee on Bankruptcy Rules' Rules Suggestions process. Both originated as field-developed methodologies before formal adoption, and both remain externally documented at the cited authority links.

Suggestion 26-BK-3 (March 2026): Automated Section 1328(f) discharge-bar screening at the point of case initiation. U.S. Courts Rules Suggestions index.

Suggestion 26-BK-5 (April 2026): Rule 9037 / Forms 121 and 309E1 SSN exposure remediation. Forwarded to the Standing Committee. U.S. Courts Rules Suggestions index. Background and empirical basis: research-process methodology page.

How to Cite

All Open Bankruptcy Project methodology contributions are released under the Creative Commons Attribution 4.0 International (CC-BY-4.0) license. Citations should reference the canonical URL of the contribution and the OBP as publisher. Suggested formats follow, using the Binary-Forcing Notice contribution as an illustrative example.

BibTeX

BibTeX
@misc{obp_binary_forcing_notice_2026,
  author       = {{Open Bankruptcy Project}},
  title        = {Binary-Forcing Notice with Conditional Variants},
  year         = {2026},
  month        = {may},
  publisher    = {Open Bankruptcy Project},
  howpublished = {\url{https://openbankruptcyproject.org/methodology/binary-forcing-notice/}},
  note         = {Methodology contribution, version 0.1.0. CC-BY-4.0.}
}

APA
