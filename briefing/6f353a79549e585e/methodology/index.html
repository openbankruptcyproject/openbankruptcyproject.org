OBP Methodology

Validation Package for Academic Review

This package contains the methodology, findings, source code, and reproduction steps for two OBP analyses. All data sources are publicly available. The code is anonymized. Nothing in this package identifies specific attorneys or cases.

1. Section 1328(f) Discharge Eligibility Analysis

Cross-references FJC IDB prior filing history against case disposition across 4.9 million cases to identify 391,951 cases where prior filers received discharge with no evidence of eligibility verification.

Methodology

Full description of the FJC cross-reference approach, data sources, filtering logic, and limitations.

View methodology.md

Findings

National summary: 391,951 cases (27.4%), policy impact, Advisory Committee citation.

View findings.md

2. Attorney Behavior Analytics

Parses PACER docket entries to measure how frequently a debtor's attorney responds to creditor motions. Sample finding: one firm responded to 30 of 309 creditor motions (9.7%).

Methodology

Docket entry classification, response matching, rate calculation, and controls.

View methodology.md

Source Code

Data Access

All analyses use publicly available data:

  • FJC Integrated Database: fjc.gov/research/idb (free download)
  • PACER docket reports: $0.10/page via CM/ECF, or free via RECAP archive
  • Live screener: 1328f.com (client-side, no server)
  • Research platform: 1328f.org

The GitHub repository with all source code is available on request: github.com/openbankruptcyproject/obp-methodology (private, collaborator access required).
